#Práctica de automatización con GitHub Actions
##Objetivos
Los objetivos de este ejercicio son:

- Conocer como configurar flujos de trabajo automatizados con *GitHub Actions*.
- Entender el funcionamiento básico de un flujo de trabajo (acontecimientos, jobs y pasos).
- Adaptar un flujo de trabajo a las necesidades de un proyecto concreto.

##Desarrollo
Se va a desarrollar un flujo para generación automática de documentación. Básicamente, en la carpeta doc del repositorio se crearán los archivos de documentación en formato Markdown y posteriormente, al realizar un *push* se realizará una generación de la documentación en formato html a través de *MkDocs* que se desplegará en *GitHub Pages*.
